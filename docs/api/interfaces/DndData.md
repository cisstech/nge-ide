# DndData

`interface`

Representation of a data shared with a drag and drop action.

## Properties

| Name | Type | Description |
| --- | --- | --- |
| `dst?` | `string` | the id of the destination node in the dom |
| `file?` | `File` | the file dragged from the file system |
| `src?` | `string` | the id of the source node in the dom |
